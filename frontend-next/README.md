frontend-next — Next.js public site

This folder contains the Next.js public website. It is scaffolded with TypeScript and minimal pages for Phase 1.

To run locally:
1. cd frontend-next
2. npm install
3. npm run dev

Ports: default 3000

Notes:
- Update NEXT_PUBLIC_SITE_URL in .env before deploying.
- Configure Cloudinary and API URLs when backend is ready.
