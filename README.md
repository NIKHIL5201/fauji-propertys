Fauji Propertys — Real Estate Management Platform

Overview
--------
This repository will contain a production-ready Real Estate Management Platform for "Fauji Propertys" (Kanpur). The platform consists of:

- frontend-next: Next.js (React + TypeScript) public customer site
- admin-spa: React + Vite + TypeScript admin dashboard
- backend-api: Node.js + Express + TypeScript REST API

Goals
-----
- Owner must never edit source code to manage content
- Admin UI for full property and lead management
- SEO-first public site with pre-rendered property pages
- Secure authentication, role-based access, and media uploads (Cloudinary)

How to proceed locally
----------------------
1. Install Node.js (v18+ recommended) and Git.
2. Clone the repository (or create a new GitHub repo and push this folder).
3. Run scaffolding scripts (to be added) to generate apps and install dependencies.

If you want the code pushed to GitHub automatically, provide the repository name and confirm permission to create a repo. Otherwise, I'll scaffold locally here and provide instructions for you to push.
